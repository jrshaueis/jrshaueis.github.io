Portfolio Work

Here are some examples of work that I have done.

Data Visualization using CIRCOS in Galaxy

• CIRCOS is a software package used for the visualization of data in a circular layout. This was used as part of the GALAXY web platform, a resource that provides robust tools for biological data analysis.
• This visualization was performed for the paper Mapping the Cell-Membrane Proteome to the Cancer Hallmarks
• The paper used mass spectrometry methods to quantify and characterize the SKBR3 cell-membrane proteome.
• Identified quantities of protein were then mapped spatially to gene positions on the chromosome.
• Knowing protein function, each protein was then categorized based on it's involvement in 10 key cancer hallmarks.
• The protein-protein interaction network seen on the inside of the circle represents STRING interaction scores ≥0.995.
• Overall, this project taught me significant data visualization skills, not only in CIRCOS, but in R, through ggplot2, seaborn, and matplotlib.

Creation of a mutated peptide database for enabling the detection of single point mutations in mass spectrometry data

• This database was created for the paper Haueis, J.R.S., Lazar, I.M. XMAn v3 - A Database of Homo sapiens Mutated Peptides (in preparation).
• The database contains several million mutated peptide entries for use in mass spectrometry analysis.
• Pending publication. Subsequent code, entry breakdowns, and other results will be posted here soon.
In doing this project, I gained several valuable skills such as:
• How to engage in data extraction, mapping, normalization, and transformation from public databases.
• How to interface with, generate, and apply FASTA formatted data.
• How to use python, particularly PANDAS, to transform data while maintaining high fidelity.
